# Lone Wolf Choose Your Own Adventure

The purpose of this project is to create a Python FastAPI web server that lets people create an account, log in, and play the Lone Wolf choose your own adventure series books. It supports multiple characters at different points in the story, so a player can register and play the story different ways. Every decision that each character makes is kept track of, and all of the book keeping of the game is done automatically, so all the player needs to do is make choices. It allows the player to look at the character sheet and history for each character they are playing as well.

## Project Standards
- Use `uv` for package management, and for a virtual env when doing any execution, testing, etc
- Maintain comprehensive testing
- Bug handling
  - Create a test covering this failure
  - Fix the bug
  - See if the test now passes
- Focus on API first
